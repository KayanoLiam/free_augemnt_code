6.2 题

问题: 写出 C₅H₁₁Br 的所有异构体，用系统命名法命名，注明伯、仲或叔卤代烃。如有手性碳原子，以“*”标出。

解答:
首先，我们画出戊烷 (C₅) 的三种碳架异构体，然后将溴原子 (Br) 放在不同的位置上。

正戊烷 (n-pentane) 碳架: CH₃-CH₂-CH₂-CH₂-CH₃

1-溴戊烷: CH₃CH₂CH₂CH₂CH₂Br (伯卤代烃)

2-溴戊烷: CH₃CH₂CH₂*CH(Br)CH₃ (仲卤代烃, C2是手性碳)

3-溴戊烷: CH₃CH₂CH(Br)CH₂CH₃ (仲卤代烃)

异戊烷 (isopentane) / 2-甲基丁烷碳架: (CH₃)₂CH-CH₂-CH₃

1-溴-2-甲基丁烷: (CH₃)₂*CH-CH(Br)CH₃ (写错了，应该是(CH₃)₂CH-CH₂Br，不对，是1-位取代) (CH₃)₂CHCH₂CH₂Br，应该是 1-溴-3-甲基丁烷: (CH₃)₂CHCH₂CH₂Br，不对，结构是 (CH₃)₂CH-CH₂-CH₃。

让我们重新梳理一下异戊烷的取代：

1-溴-2-甲基丁烷: (CH₃)₂CH-CH(Br)CH₂CH₃ (应该是 2-溴-3-甲基丁烷: CH₃CH(Br)*CH(CH₃)CH₃ (仲卤代烃, C2, C3 都是手性碳))

1-溴-2-甲基丁烷: BrCH₂*CH(CH₃)CH₂CH₃ (伯卤代烃, C2是手性碳)

2-溴-2-甲基丁烷: (CH₃)₂C(Br)CH₂CH₃ (叔卤代烃)

1-溴-3-甲基丁烷: BrCH₂CH₂CH(CH₃)₂ (伯卤代烃)

新戊烷 (neopentane) / 2,2-二甲基丙烷碳架: C(CH₃)₄

1-溴-2,2-二甲基丙烷: BrCH₂C(CH₃)₃ (伯卤代烃)

总共有 8 种结构异构体。其中有手性碳的包括：

2-溴戊烷

1-溴-2-甲基丁烷

2-溴-3-甲基丁烷

6.3 题

问题: 写出二氯代的含四个碳的烷烃的所有异构体... 如有手性碳原子，以“*”标出。

解答:
碳架有两种：正丁烷和异丁烷。

正丁烷 (n-butane) 碳架:

1,1-二氯丁烷: Cl₂CHCH₂CH₂CH₃

1,2-二氯丁烷: ClCH₂*CH(Cl)CH₂CH₃ (有手性碳)

1,3-二氯丁烷: ClCH₂CH₂*CH(Cl)CH₃ (有手性碳)

1,4-二氯丁烷: ClCH₂CH₂CH₂CH₂Cl

2,2-二氯丁烷: CH₃C(Cl₂)CH₂CH₃

2,3-二氯丁烷: CH₃*CH(Cl)*CH(Cl)CH₃ (有手性碳, 存在内消旋体和对映异构体)

异丁烷 (isobutane) / 2-甲基丙烷碳架:

1,1-二氯-2-甲基丙烷: Cl₂CHCH(CH₃)₂

1,2-二氯-2-甲基丙烷: ClCH₂C(Cl)(CH₃)₂

1,3-二氯-2-甲基丙烷: ClCH₂CH(CH₃)CH₂Cl

总共有 9 种结构异构体。

6.4 题

问题: 写出下列反应的主要产物。

解答:

a. 这是格氏试剂的制备和应用的反应。

C₆H₅CH₂Cl + Mg → C₆H₅CH₂MgCl (氯化苄基镁)

C₆H₅CH₂MgCl + CO₂ → C₆H₅CH₂COOMgCl

C₆H₅CH₂COOMgCl + H⁺/H₂O → C₆H₅CH₂COOH (苯乙酸)

b. 这是一个消除反应。强碱 NaOC₂H₅ 作用下，偕二卤代烷发生两次消除反应，生成炔烃。

产物: CH₃-C≡CH (丙炔)

c. 这是一个 E2 消除反应。KOH/乙醇是强碱，加热有利于消除。根据扎伊采夫规则（Zaitsev's rule），双键主要生成在与取代基相连的、被取代基更多的碳原子之间。

产物: 1-甲基环己烯

d. 这是一个消除反应。底物是1-氯-2-苯基乙烷。在强碱（NaOH）和加热条件下，主要发生 E2 消除。生成的烯烃与苯环共轭，非常稳定。

产物: C₆H₅-CH=CH₂ (苯乙烯)

e. 这是格氏试剂与活泼氢化合物（乙醇）的反应。

CH₃CH₂CH₂CH₂Br + Mg → CH₃CH₂CH₂CH₂MgBr (正丁基溴化镁)

CH₃CH₂CH₂CH₂MgBr + C₂H₅OH → CH₃CH₂CH₂CH₃ (丁烷) + Mg(Br)OC₂H₅

f. 这是一道关于 SN2 反应的题目。底物 H₃C-CH(CH₃)-CH₂Cl (1-氯-2-甲基丙烷) 是一个伯卤代烃，空间位阻小，有利于 SN2 反应。OH⁻ 作为亲核试剂进攻，Cl⁻ 作为离去基团。

产物: H₃C-CH(CH₃)-CH₂OH (2-甲基-1-丙醇)

g. 这是一道关于 SN1 反应的题目。底物是叔卤代烃，在极性质子溶剂（水）中发生 SN1 反应。首先离去基团 Cl⁻ 离去形成稳定的叔碳正离子，然后亲核试剂 H₂O 进攻，最后失去一个质子。

产物: 1-甲基-1-环己醇

h. 这是一个从烯烃到卤代烷，再回到烯烃的反应。从环己烯到溴代环己烷是 HBr 的亲电加成反应。从溴代环己烷变回环己烯是一个消除反应，需要强碱。

所需的试剂: KOH/乙醇, 加热 (或其他强碱，如 NaNH₂ 或 t-BuOK)

i. 这是一个 SN2 反应。烯丙基氯 (Allyl chloride) 是一个活泼的卤代烃，CN⁻ 是一个很好的亲核试剂。

产物: CH₂=CH-CH₂-CN (3-丁烯腈)

j. 叔卤代烃 (CH₃)₃CCl 在强碱（NaOH）作用下，主要发生 E2 消除反应。

产物: (CH₃)₂C=CH₂ (异丁烯 或 2-甲基丙烯)

6.5 题

问题: 下列各对化合物按 SN2 机理进行反应，哪一个反应速率较快？说明原因。

解答:
SN2 反应速率主要受空间位阻影响：伯卤代烃 > 仲卤代烃 > 叔卤代烃。

a. CH₃CH₂CH₂CH₂Cl (伯) vs CH₃CH₂CH(Cl)CH₃ (仲)

CH₃CH₂CH₂CH₂Cl (1-氯丁烷) 更快。

原因: 1-氯丁烷是伯卤代烃，反应中心碳原子的空间位阻较小，亲核试剂更容易从背面进攻。2-氯丁烷是仲卤代烃，空间位阻较大。

b. CH₃CH₂CH₂CH₂Cl vs CH₃CH₂CH=CHCl (乙烯基卤)

CH₃CH₂CH₂CH₂Cl (1-氯丁烷) 更快。

原因: 乙烯基卤代烃（卤素连在双键碳上）的 C-Cl 键是 sp² 杂化碳，键能更强，不易断裂。同时，双键的 π 电子云和分子结构阻碍了亲核试剂的背面进攻。因此，乙烯基卤代烃极难发生 SN2 反应。

c. C₆H₅Br (芳基卤) vs C₆H₅CH₂Br (苄基卤)

C₆H₅CH₂Br (苄基溴) 更快。

原因: 芳基卤代烃（卤素直接连在苯环上）的 C-Br 键是 sp² 杂化碳，键能强，且苯环阻碍了背面进攻，因此不发生 SN2 反应。而苄基溴是伯卤代烃，其 SN2 反应的过渡态可以与苯环的 π 体系共轭，从而得到稳定，反应速率很快。

6.6 题

问题: 将下列化合物按 SN1 机理反应的活性由大至小排列。

解答:
SN1 反应速率取决于生成碳正离子的稳定性。稳定性顺序：叔 > 仲 > 伯。同时，离去基团的能力越强（对应酸的酸性越强），反应越快 (I⁻ > Br⁻ > Cl⁻ > F⁻)。

a. (CH₃)₂CHBr: 仲卤代烃，生成仲碳正离子。

b. (CH₃)₃CI: 叔卤代烃，生成叔碳正离子，离去基团是 I⁻。

c. (CH₃)₃CBr: 叔卤代烃，生成叔碳正离子，离去基团是 Br⁻。

比较:

(b) 和 (c) 都能生成稳定的叔碳正离子，比 (a) 生成的仲碳正离子稳定得多，所以 (b) 和 (c) 的活性远大于 (a)。

比较 (b) 和 (c)，它们生成相同的碳正离子，但离去基团不同。I⁻ 是比 Br⁻ 更好的离去基团（因为 HI 的酸性比 HBr 强）。

因此，(b) 的反应速率最快。

排列顺序: (b) > (c) > (a)
即: (CH₃)₃CI > (CH₃)₃CBr > (CH₃)₂CHBr

6.8 题

问题: 分子式为 C₄H₉Br 的化合物 A, 用强碱处理, 得到两个分子式为 C₄H₈ 的异构体 B 及 C。写出 A, B, C 的结构式。

解答:
这是一个消除反应，生成了两种不同的烯烃异构体，说明底物 A 应该有不止一种 β-氢可以被消除。

A 的结构: 可能的 C₄H₉Br 异构体有 1-溴丁烷、2-溴丁烷、1-溴-2-甲基丙烷、2-溴-2-甲基丙烷。

1-溴丁烷消除只生成 1-丁烯。

1-溴-2-甲基丙烷和 2-溴-2-甲基丙烷消除都只生成异丁烯。

只有 2-溴丁烷 进行消除反应时，可以得到两种产物：1-丁烯和 2-丁烯 (2-丁烯还分顺式和反式)。这符合题意。

所以 A 是 2-溴丁烷 (CH₃-CH(Br)-CH₂-CH₃)。

B 和 C 的结构: 2-溴丁烷在强碱作用下发生 E2 消除，生成扎伊采夫产物（主要）和霍夫曼产物（次要）。

扎伊采夫产物是 2-丁烯 (CH₃-CH=CH-CH₃)，这是 B 和 C 中的一个 (通常是主要产物)。

霍夫曼产物是 1-丁烯 (CH₂=CH-CH₂-CH₃)，这是 B 和 C 中的另一个。

因此:

A: CH₃CH(Br)CH₂CH₃ (2-溴丁烷)

B: CH₃CH=CHCH₃ (2-丁烯)

C: CH₂=CHCH₂CH₃ (1-丁烯)
(B 和 C 可以互换)

6.10 题

问题: 写出由 (S)-2-溴丁烷 制备 (R)-CH₃CH(OCH₂CH₃)CH₂CH₃ 的反应机理。

解答:
这个反应中，底物的构型是 S 型，产物的构型是 R 型，发生了构型翻转。这是 SN2 反应 的典型特征。反应物应该是乙醇钠 (NaOCH₂CH₃)，提供强亲核试剂乙氧基负离子 (CH₃CH₂O⁻)。

反应机理 (SN2):

亲核进攻: 亲核试剂 CH₃CH₂O⁻ 从与 C-Br 键相对的背面进攻 (S)-2-溴丁烷的手性碳原子。

过渡态: 形成一个五配位的过渡态。在这个过渡态中，C-O 键正在形成，C-Br 键正在断裂。手性碳上的三个非反应基团 (H, -CH₃, -CH₂CH₃) 近似处于一个平面上。

构型翻转: C-Br 键完全断裂，Br⁻ 作为离去基团离开。同时 C-O 键完全形成。整个分子的构型像一把被风吹翻的雨伞，发生 瓦尔登翻转 (Walden Inversion)，从 S 构型变为 R 构型。

图示 (可以用楔形/虚线表示立体化学):

OCH₂CH₃
        |
  EtO⁻  +   H--C--Br     --->  [ EtO---C---Br ]⁻  --->  EtO--C--H  +  Br⁻
            /  \                / | \                       / \
          CH₃  C₂H₅             H CH₃ C₂H₅                 CH₃ C₂H₅
         (S)-构型             (过渡态)                    (R)-构型


(注: Et 代表乙基 -CH₂CH₃, C₂H₅)

6.12 题

问题: 由 2-甲基-1-溴丙烷及其他无机试剂制备下列化合物。

解答:
起始物: (CH₃)₂CHCH₂Br (2-甲基-1-溴丙烷)

a. 异丁烯: (CH₃)₂C=CH₂

通过 E2 消除反应。

反应: (CH₃)₂CHCH₂Br + KOH (乙醇溶液, 加热) → (CH₃)₂C=CH₂

b. 2-甲基-2-丙醇: (CH₃)₃COH

两步法：先制备异丁烯，再进行亲电加成。

步骤1: (CH₃)₂CHCH₂Br + KOH (乙醇, Δ) → (CH₃)₂C=CH₂ (如 a)

步骤2: (CH₃)₂C=CH₂ + H₂O (在稀 H₂SO₄ 催化下) → (CH₃)₃COH (酸催化水合，遵循马氏规则)

c. 2-甲基-2-溴丙烷 (应为 2-溴-2-甲基丙烷): (CH₃)₃CBr

两步法：先制备异丁烯，再加成 HBr。

步骤1: (CH₃)₂CHCH₂Br + KOH (乙醇, Δ) → (CH₃)₂C=CH₂ (如 a)

步骤2: (CH₃)₂C=CH₂ + HBr → (CH₃)₃CBr (亲电加成，遵循马氏规则)

d. 1,2-二溴-2-甲基丙烷: (CH₃)₂C(Br)CH₂Br

两步法：先制备异丁烯，再与溴进行加成。

步骤1: (CH₃)₂CHCH₂Br + KOH (乙醇, Δ) → (CH₃)₂C=CH₂ (如 a)

步骤2: (CH₃)₂C=CH₂ + Br₂ (在 CCl₄ 等非质子溶剂中) → (CH₃)₂C(Br)CH₂Br

e. 1-溴-2-甲基-2-丙醇: HO-C(CH₃)₂-CH₂Br

两步法：先制备异丁烯，再在水中与溴反应生成卤代醇。

步骤1: (CH₃)₂CHCH₂Br + KOH (乙醇, Δ) → (CH₃)₂C=CH₂ (如 a)

步骤2: (CH₃)₂C=CH₂ + Br₂/H₂O → HO-C(CH₃)₂-CH₂Br (亲电加成，OH 加在能形成更稳定碳正离子的碳上，即叔碳)

6.13 题

问题: 分子式为 C₃H₇Br 的 A, 与 KOH-乙醇溶液共热得 B, 分子式为 C₃H₆。如使 B 与 HBr 作用，则得到 A 的异构体 C。推断 A 和 C 的结构, 用反应式表明推断过程。

解答:

分析 A 到 B: A (C₃H₇Br) 在 KOH/乙醇/加热条件下生成 B (C₃H₆)，这是典型的 E2 消除反应。C₃H₆ 唯一的结构是 **丙烯 (CH₃-CH=CH₂) **。所以 B 是丙烯。

分析 B 到 C: B (丙烯) 与 HBr 反应得到 C。这是 HBr 对烯烃的亲电加成反应，遵循 马尔科夫尼科夫规则 (Markovnikov's rule)，即 H 加到含氢较多的双键碳上，Br 加到含氢较少的双键碳上。

CH₃-CH=CH₂ + HBr → CH₃-CH(Br)-CH₃ (2-溴丙烷)

所以 C 是 2-溴丙烷。

推断 A: 题目说 C 是 A 的异构体。我们已经知道 C 是 2-溴丙烷。C₃H₇Br 只有两种异构体：1-溴丙烷和 2-溴丙烷。既然 C 是 2-溴丙烷，那么 A 必定是它的异构体。

所以 A 是 1-溴丙烷 (CH₃CH₂CH₂Br)。

验证:

A (1-溴丙烷) 消除生成 B (丙烯)，正确。

B (丙烯) 与 HBr 反应生成 C (2-溴丙烷)，正确。

C (2-溴丙烷) 是 A (1-溴丙烷) 的同分异构体，正确。

反应式:

A 的结构: CH₃CH₂CH₂Br

C 的结构: CH₃CH(Br)CH₃

推断过程:

CH₃CH₂CH₂Br     + KOH/乙醇, Δ  ---->   CH₃-CH=CH₂   + KBr + H₂O
      (A)                               (B)

  CH₃-CH=CH₂     + HBr            ---->   CH₃-CH(Br)-CH₃
      (B)                               (C, A的异构体)
